# General rules

Compile your code with </br>
`c++` `-Wall -Wextra -Werror` `-std=c++98` `-pedantic` </br>

Write class names in UpperCamelCase format (CPP is case sensitive)

⛔️ `using namespace <ns_name>`

⛔️ `friend`

⛔️ `*printf()`, `*alloc()`, `free()`

Detailed README in ex0X folders
